# 智能储物柜系统

基于 STM32F103C8 的智能储物柜嵌入式控制系统。

## 功能

- 温湿度检测（DHT11）
- OLED 显示（温度、湿度、电压等信息）
- 触控交互与菜单系统
- 电机与舵机控制
- 超声波测距
- 蜂鸣器提示
- 串口数据输出

## 技术栈

- **主控**: STM32F103C8 (Cortex-M3)
- **开发环境**: Keil MDK (uvprojx)
- **外设驱动**: GPIO、I2C、PWM、ADC、Serial、Timer

## 目录结构

```
User/       — 主程序逻辑
Hardware/   — 外设驱动代码 (OLED, DHT11, 电机, 舵机, 触摸等)
Library/    — STM32 标准外设库
Start/      — 启动文件
System/     — 系统配置
```

## 硬件清单

- STM32F103C8 最小系统板
- 0.96" OLED 显示屏 (I2C)
- DHT11 温湿度传感器
- 舵机 SG90 × 若干
- 直流电机 + 驱动模块
- HC-SR04 超声波模块
- 触摸按键模块
- 蜂鸣器模块
